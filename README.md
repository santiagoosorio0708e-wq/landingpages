## Estructura típica de cada landing

```text
N/
├── index.html
├── CSS/ o css/
│   └── style.css o styles.css
└── media/
    └── (imágenes y recursos)
```

## Cómo visualizar las landings en local

1. Clona este repositorio.
2. Abre la carpeta del proyecto.
3. Entra a la carpeta de la landing que quieras ver.
4. Abre el archivo `index.html` en el navegador.

También puedes usar un servidor local (recomendado):

```bash
# Desde la raíz del proyecto
python3 -m http.server 5500
```
